Portfólio pessoal desenvolvido com **Vue 3**, **Tailwind CSS** e **Vite**, apresentando minha trajetória como Desenvolvedor Full Stack. O projeto conta com animações suaves ao rolar a página (AOS), tema claro/escuro, seções de sobre mim, experiências profissionais, projetos e contato.

![Preview do site](https://iago-dev.vercel.app/) *(substitua pelo seu link real)*

## ✨ Funcionalidades

- 🌓 **Tema claro/escuro** com persistência em `localStorage`
- 🎬 **Animações de entrada ao rolar** usando [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- 📱 **Design totalmente responsivo** (mobile first)
- ⚡ **Performance otimizada** com Vite
- 🎯 **Seções**:
  - **Hero** com foto, chamada e links sociais
  - **Sobre mim** com resumo profissional e marquee interativo de tecnologias (ícones do Simple Icons)
  - **Experiência profissional** em timeline interativa
  - **Projetos** em cards com estatísticas e links para demo/github
  - **Rodapé** com redes sociais e copyright

## 🛠️ Tecnologias Utilizadas

- **Vue 3** (Composition API, `<script setup>`)
- **Vite** (build rápido e desenvolvimento)
- **Tailwind CSS 3** (estilização utilitária)
- **AOS** (animações de scroll)
- **Lucide Icons** (ícones principais)
- **Simple Icons** (ícones das tecnologias no marquee)
- **vue-simple-icons** (integração dos ícones no Vue)
