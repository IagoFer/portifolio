## 🧠 Sobre o Projeto

Este portfolio foi construído como uma aplicação moderna orientada à experiência do usuário e apresentação profissional, priorizando:

- Performance e fluidez visual
- Narrativa técnica e profissional
- Demonstração de impacto real dos projetos
- Arquitetura frontend escalável

O objetivo não é apenas exibir projetos, mas demonstrar **capacidade de engenharia, liderança técnica e entrega de produtos em produção**.

---

## ✨ Funcionalidades

### 🌙 Experiência de Interface
- 🌓 Dark Mode padrão com persistência em `localStorage`
- 🎨 Transição suave entre temas
- 📱 Layout totalmente responsivo (Mobile First)

### 🎬 Motion & UX
- Animações de entrada ao rolar usando **AOS**
- Microinterações em botões e cards
- Hero dinâmico com frases rotativas
- Transições suaves entre componentes

### 🎯 Estrutura do Portfólio
- **Hero** — posicionamento profissional
- **Sobre mim** — resumo técnico
- **Experiência** — timeline profissional orientada a resultados
- **Projetos** — produtos com métricas de impacto
- **Footer** — hub de contato

## 🏗️ Arquitetura do Projeto
