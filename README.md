Portfólio pessoal desenvolvido com **Vue 3**, **Tailwind CSS** e **Vite**, apresentando minha trajetória como Desenvolvedor Full Stack. O projeto conta com animações suaves, tema claro/escuro, seções de sobre mim, experiências profissionais, projetos e contato.

![Preview do site](https://via.placeholder.com/1200x600?text=Preview+do+Portfólio)

## ✨ Funcionalidades

- 🌓 **Tema claro/escuro** com persistência em `localStorage`
- 🎬 **Animações de entrada e scroll** usando `@vueuse/motion`
- 📱 **Design totalmente responsivo** (mobile first)
- ⚡ **Performance otimizada** com Vite
- 🧩 **Componentes reutilizáveis** (Parallax, ImageWithFallback, etc.)
- 🎯 **Seções**:
  - Hero com foto e chamada
  - Sobre mim (resumo profissional e skills)
  - Experiência profissional (timeline interativa)
  - Projetos (cards com estatísticas e links)
  - Rodapé com redes sociais

## 🛠️ Tecnologias Utilizadas

- **Vue 3** (Composition API, `<script setup>`)
- **Vite** (build rápido e desenvolvimento)
- **Tailwind CSS 4** (estilização utilitária)
- **@vueuse/motion** (animações declarativas)
- **@vueuse/core** (utilitários reativos)
- **lucide-vue-next** (ícones modernos)
